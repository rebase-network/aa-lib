# GameOn 基本场景

通过 EIP4337 标准，新 Web3 用户很容易通过这种方法进来，并获得第一个个人钱包。

## paymaster 提供 WETH/USDX 支付 gas 费用

支持使用非原生 ETH 支付 gas 费用，这里可选 WETH，或者通过 Oracle 或者 USD 和 ETH 之间价格，可以让用户用 USDX 支付 gas 费用

## gasless 项目方代付 gas 费用

在前期引流或者其他白名单场景，项目方可以用 gasless 来交易，通过 bundler 代付，用户可以使用 4337 顺滑流程并且无成本玩 Dapp

## 使用项目方 Token 支付 gas 费用

用户可以直接进来玩，不需要记录私钥，从 Free2Play 入手，
用户不需要支付费用，通过付出时间，积累到一定程度就可以激活钱包，
项目方收获用户的同时，token 也能获得初始使用场景

## 打通游戏方和各大 AA 钱包客户端

通过兼容 4337 合约和其他钱包合作，使用我们 SDK 的用户能直接接入其他 aa 钱包客户端，打破孤岛
使用我们钱包获得游戏角色 SBT 和装备 NFT，都可以直接应用到下个游戏或展示上
